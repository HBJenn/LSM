# Lengua de Señas Mexicana (LSM)

LSM (Lengua de Señas Mexicana) es un proyecto que usa inteligencia artificial para identificar y clasificar los gestos de la lengua de señas, especialmente las letras del abecedario (A-Z) y las vocales (A, E, I, O, U). Utilizando la plataforma Teachable Machine, la idea es hacer más fácil aprender y comunicarse en LSM, combinando tecnología accesible con una propuesta de inclusión educativa. 
Este proyecto tiene dos modelos entrenados por separado: 1. Modelo del alfabeto (26 letras): que reconoce las señas estáticas de cada letra. 2. Modelo de vocales (5 letras): enfocado en detectar las vocales con mayor precisión, ideal para quienes están empezando. Para entrenarlos, se tomaron fotos de las posiciones de las manos, usando una red neuronal sencilla para categorizarlas. Así, el sistema puede interpretar las señas en tiempo real, usando una cámara, y mostrar la letra en pantalla. 
¿Por qué y para qué? La idea nació del interés personal por aprender LSM y de explorar cómo la inteligencia artificial puede mejorar la comunicación. Entre las aplicaciones posibles están: Una herramienta para que los principiantes aprendan LSM.  Un prototipo para traducción básica de señas a texto. Y posiblemente a futuro un sistema donde reconozca letras e incluso palabras que se puedan traducir a voz. 
Un poco sobre cómo funciona el aprendizaje de máquina en este proyecto: Este método permite que la máquina aprenda a reconocer patrones visuales, como la posición de los dedos y mano 
Teachable Machine hace que esto sea más sencillo, ya que usa una interfaz fácil de entender, y en el fondo emplea algoritmos de clasificación supervisada, que aprenden con ejemplos ya etiquetados. En el contexto de la LSM, esto significa que podemos crear una tecnología accesible para promover la inclusión social, mostrando que la inteligencia artificial también puede contribuir en la inclusión social. No poder describir artísticamente su uso, sin embargo, en mi profesión (Educación) es una buena herramienta para poder aplicar la tecnología en las aulas, que no sea de una forma ambigua si no que logre un impacto en el proceso de aprendizaje de los individuos.
